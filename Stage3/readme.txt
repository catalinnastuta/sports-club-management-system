=======================================================================
User Guide: How to Compile and Run the Java Program - Catalin & Dominic
=======================================================================

1. System Requirements:
   - Java Development Kit (JDK) installed on your computer.
   - A text editor or an Integrated Development Environment (IDE) like IntelliJ IDEA, Eclipse, or NetBeans.

2. Compiling the Program:
   a. Open a terminal or command prompt.
   b. Navigate to the directory containing the Java source code file (`Main.java`).
   c. Use the following command to compile the program:
      ```
      javac Main.java
      ```

3. Running the Program:
   a. After successful compilation, run the program using the following command:
      ```
      java Main
      ```

4. Using the Program:
   - The program will display a text-based menu interface with numbered options.
   - Enter the number corresponding to the action you want to perform and press Enter.
   - Follow the on-screen prompts to navigate through menus and perform tasks such as managing clients, memberships, competitions, and staff.

5. Exiting the Program:
   - To exit the program, choose the "Exit" option from the main menu.

6. Handling Errors:
   - The program includes error handling for invalid input. If you encounter an error, follow the on-screen instructions to rectify it.

7. Data Persistence:
   - Data for clients, memberships, competitions, and staff is stored in separate files (`ClientFile`, `MembershipFile`, `CompetitionFile`, `StaffFile`) and will persist between program executions.
